Machine Learning Tasks

During the module, I engaged in a variety of machine learning (ML) tasks to enhance my knowledge and understanding of key concepts. These included:

  • Exploratory Data Analysis (EDA): Gaining insights into datasets through visualization and descriptive statistics.
  • Covariance and Pearson Correlation: Exploring relationships between variables to understand dependencies and patterns.
  • Linear Regression: Modeling relationships between dependent and independent variables for predictive analysis.
  • Multiple Linear Regression: Extending linear regression to include multiple predictors for more complex datasets.
  • Polynomial Regression: Capturing nonlinear relationships between variables for better predictive performance.
  • Linear Regression with Scikit-Learn: Utilizing Python libraries to implement and evaluate regression models.
  • Correlation Analysis: Assessing the strength and direction of relationships between variables.
  • Clustering: Grouping data into meaningful clusters to identify patterns and similarities.
  • Jaccard Coefficient: Measuring similarity between datasets or clusters.
  • K-Means Clustering: Partitioning data into clusters based on similarity for unsupervised learning.

These tasks provided a strong foundation in ML techniques.
